# Flourish 2

I tried to highlight the lack of data given in the dataset Street Closures, which was created by the Seattle Department of Transportation. What the dataset was actually doing was a bit divorced from that all encompassing title. For one, as the title of my visual quotes, the dataset does not include certain types of street closures including for construction, which seems like it would be a major use case for this type of dataset. Additionally, a good chunk of the 88 street closures that are actually listed don't have a descriptive label, causing them to be lumped in Temporary Activiation.
